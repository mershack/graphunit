GraphUnit Tutorial

An Example Using only Images without a dataset

  1. Download and extract this viewer files to your computer. Click here to download
  2. Go to the Userstudy Setup Page
  3. To add Viewers: From the Study Setup Page, Click on the "Create or Update Viewer Directories" button.
    To add a new viewer directory, specify a short name for the viewer. e.g. myGraphViewer
    Upload files: Click on the choose files button and go into the directory where you extracted the downloaded files. Select all the files in that directory and submit the uploaded files
  4. If you have an introduction file, let your html viewer files return the name of the introduction file using the function "setIntroduction". (See an example of how that function is implemented in the bw_viewer.html and colored_viewer.html files)
  5. Create Tasks: From the Study Setup Page, Click on the "Create or Update Tasks" button.
    Provide a short name (one-word) for the task. e.g. findMyWaldo
    Provide the task question. e.g. Can you see Waldo in the Picture?
    Under the Answer type section, Select "yes" for will this task have correct answers. And for answer type, select Options(fixed). And specify the following options: Yes, No, Maybe
  6. Upload TaskInstance: From the Study Setup Page, click on the "Upload TaskInstance" button.
    In the quantitative tasks section, select the task you created, from the drop down list. It should be the last option in that list.
    Upload the task instance. Download and use this example. Find Waldo Task Instance Example
    The task instance is specifies, the input, and answers for each task instance.
  7. Now you can create a study. On the Study Setup Page, In the Viewers section, select the viewer directory you created from among the list. (Refresh the page, if you can't find it).
    For condition URL, select bw_viewer.html from the list
    For the second viewer, select colored_viewer.html from the list
  8. In the Experiment Design Type for the visualizations section, select Between-subjects from the options
  9. In the Actual Study section, select the task you created from the list. And then specify the number of questions and time for each question (NB. Enter 0 for time, if you do not want the study to be timed). Make sure the number of questions you specify (plus the number of training instances) is not more than the number of task instances you have.
  10. Click on the Demo button to create a demo of the study.
    When you are satisfied with the demo, you can click on the save study to save the study.
